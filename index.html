Basic Syntax

types of data:

-string: a string is a piece of text surrounded by quotation marks, they can be double or single, but you can't mix them.

-integer: an integer is a number that is not surrounded by quotes

-float: a float is a number with a decimal value, not surrounded by quotes

-boolean: a boolean data is either True or False.

when naming variables the name goes first then an '=' sign and then the value.
